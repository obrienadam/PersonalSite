LATTE Lightweight Applications for Temporal Transport Equations

About

Lightweight Applications for Temporal Transport Equations (LATTE) is a lightweight Python based 2D multiphase CFD solver developed as part of my PhD project. The intention of this project is to develop a 2D solver with a complete graphical interface and the ability to render solution data in real time. This allows for rapid development and testing of new CFD algorithms in a simplified environment, before migrating them to a more heavy duty 3D framework such as CAFFE.

Installation

The most recent version can be found here. To clone this repository, open a terminal and type the commmand:

> git clone https://www.github.com/obrienadam/LATTE

LATTE depends on the following third-party development libaries:

  • Python 2.7.5
  • Numpy 1.10.4
  • SciPy 0.12.0
  • Matplotlib 1.2.0
  • PyQt4 4.10.1

No installation is required, simply run:

> python2 ~/latte/bin/latte.py

to open up the graphical interface and configure a case.
