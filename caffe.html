CAFFE Computational Algorithms Framework for Fluid Equations

About

Computational Algorithms Frame for Fluid Equations (CAFFE) is open source software I developed for my PhD project, designed to provide a flexible framework for solving coupled transport equations using a finite-volume approach. The software supports arbitrarily shaped hexahedral meshes, MPI block-based parallelization, and a fully implicit solution method. A Git repository for the source code can be found here. The code is still in early development, but is already able to solve a variety of incompressible fluid mechanics problems.

At the present moment, development of CAFFE is focused on implementing a direct forcing immersed boundary method for the simulation of particles interacting with fluid interfaces.

Installation

CAFFE depends on the following third-party development libaries:

  • CMake 2.8.11
  • Boost 1.53
  • Petsc 3.4
  • BLAS/LAPACK 3.5.0
  • An Message Passing Interface (MPI) implementation, eg. MPICH 3.1

The most recent version can be found here. To clone this repository, open a terminal and type the commmand:

> git clone https://www.github.com/obrienadam/CAFFE

Then, assuming that the repository was cloned into the home directory, navigate to the project directory:

> cd ~/CAFFE

and run the command

> cmake CMakeLists.txt -DCMAKE_BUILD_TYPE=Release

to generate the make files. Then, build the project with

> make all

The solver binaries will be placed into $HOME/CAFFE/Modules. In order to run a case, such as the 3D lid driven cavity case, navigate to the case directory:

> cd ~/CAFFE/ExampleCases/Cavity3D

and execute the desired solver from the case directory; for example:

> ~/CAFFE/Modules/caffePiso
