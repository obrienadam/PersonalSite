CAFFE

Computational Algorithms Framework for Fluid Equations

Computational Algorithms Framework for Fluid Equations (CAFFE) is a scalable, extensible and object oriented 3D CFD framework written in C++. This command line based application is designed to solve large scale 3D incompressible flows on distributed parallel computers using the Message Passing Interface (MPI).

Phase

Unstructured 2D Multiphase Solver

Phase is a lightweight multiphase solver for unstructured meshes written in C++. The code features several high-order compressive Volume-of-Fluid (VoF) advection algorithms for maintaining sharp fluid-fluid interfaces on unstructured meshes; an issue that has consistently proven to be problematic for researchers. Future releases of phase will also include Immersed Boundary (IB) capabilities as well. Additionally, Phase is designed to allow for rapid development and testing of new finite-volume schemes, by allowing the programmer to write partial differential equations almost as they would appear on paper.

LATTE

Lightweight Applications for Temporal Transport Equations

Lightweight Applications for Temporal Transport Equations (LATTE) is a lightweight Python based 2D CFD application with a graphical interface. The purpose of LATTE is to provide a lightweight framework for rapid development and testing of numerical methods for solving partial differential equations, in particular the governing equations of incompressible multiphase flows.
